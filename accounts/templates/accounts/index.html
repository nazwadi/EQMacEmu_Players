What is EQA?

EQA is an EQ emulator progression server that advances when selected raid targets in the current expansion are defeated. You can choose to raid as part of a guild, or participate in the weekly server-wide open raids.

What is the ruleset?

  • Firiona Vie-style Free Trade. Almost all items are tradeable!
  • Legacy item drops remain in game permanently at adjusted rarity. Items don't get nerfed.
  • Keyrings are enabled from launch, along with custom keyring settings (e.g. Plane of Sky keys go to keyring)
  • Miragul's Highway is implemented (travel by firepots)
  • Experience bonuses for the 4th, 5th, and 6th group members
  • Server launched at Kunark expansion. New expansions release approximately six months after all raid targets in the current expansion have been defeated.
  • 3-Box Limit. This is enforced by IP and GM Logs.
  • /ooc and /auction are server-wide
  • Player vs Environment (PVE)

Web Tools

The website you're at now is where you can query the live game database for information of various kinds.

Some key features we've implemented:

  • Character viewer that shows you everything about your characters (faction, inventory, spell lists including which ones you've scribed)
  • Global inventory search across all of your characters (must be logged in)
  • Self-service character transfer between game accounts associated with your EQA account (must be logged in)
  • Search items, factions, NPCs, Recipes, pets, spell lists, zones
  • Zone pages list npcs, foraged items, fishing, spawn points, etc...

How to Play

Download the Client

You can play the client on Windows, Linux, or a 32-bit MacOS machine if you still have one!

Go to https://wiki.takp.info/index.php/Getting_Started for client install instructions.

Configure eqhost.txt

Edit your client eqhost.txt file to the following: 

[Registration Servers]
{
"loginserver.eqarchives.com:6999"
}
[Login Servers]
{
"loginserver.eqarchives.com:6000"
}
Create EQA Accounts
  1. First create an EQA Web Account, or Login if you already have one.
  2. Once you've logged in, go to the Accounts page and create your first Game Account. This is the account you will use to login to the game from the EQ client.
Join us on Discord!

EQA is all about a community of EQ fans many of whom have been playing since beta. We use the discord to get server announcements, various updates, talk strategy, post auctions when not in-game, voice chat during open server raids, post raid announcements, post bug reports, etc...
