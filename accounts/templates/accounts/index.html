Welcome! This app will allow you to manage your login server (aka game) accounts.

To get started, select the "Accounts" link at the top to create an account, or follow the client setup instructions below.

Client Setup

Installing the Client

Go to https://wiki.takp.info/index.php/Getting_Started for client install instructions.

Configure eqhosts.txt

Edit your client eqhosts.txt file to the following: 

            [Registration Servers]
            {
            "loginserver.eqarchives.com:6999"
            }
            [Login Servers]
            {
            "loginserver.eqarchives.com:6000"
            }

Create your Login Accounts

Go to Accounts or select it from the navbar on the top of this page.

Once you have installed your client, configured eqhosts.txt, and created a login account, you're all set to play the game.
