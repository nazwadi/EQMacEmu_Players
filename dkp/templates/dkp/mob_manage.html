Mob Roster

Mobs here are the circuit's permanent roster. Add each raid boss once — they'll be available to select on every future raid.

Adding Mobs
  • Count — adds multiple copies with auto-numbered names. Use this when two instances of the same mob exist with different loot tables (e.g., TVX #1, TVX #2).
  • DKP Value — how much DKP members earn when this mob is killed. Can be changed later; the change history is recorded.
Linking an NPC

Linking a mob to a game database NPC enables the Quick Award from Drops feature in the raid Loot tab — it pulls the mob's actual drop table so you can award loot with one click.

  • Search by partial NPC name. Only NPCs with a configured loot table will appear.
  • If a mob doesn't show up, it may not have a loot table set up in the game database.
  • Linking is optional — mobs without an NPC link still work for DKP awarding.
Deactivating

Deactivating a mob hides it from future raids but preserves all historical data. You can reactivate it at any time.
